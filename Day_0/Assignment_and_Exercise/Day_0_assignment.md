=>> Pattern Reconfition mane problem porei buje fela ei problem ta kon pattern er modde pore, jodi ami pattern somporke idea rakhte pari tahole ct te ashole beshi dhoroner problem nai. Matro koyekta pattern ba dhroner problem e beshi ashe, Ek maximum er problem e vibinno vabe aste pare, jemon : Highest, Longest, Largest, Maximum highest , etc. Ekhon amake emon vabe train korte hobe jate ami problem dhekhei bujte pari oh... eita to Maximum er pattern ta!

4. Brute Force কী?
=>> Brute Force holo emon ekta pddoti jeikhane amara ekta problem solve korar jonno joto gula somvabona ache sob e check kore dekhi, ekta o bad dei na, eita onek somoy sapekho but eita korle code bhul hpyar somvabona kom thake , code korte easy hoy, cocept valo vabe dhora jay. Amra kono problem solve korar khetre age brute force kore code ses kore then optimize korar chesta korbo tahole porblem ta khub valo vabe and easy solve korte parbo. Amra jani je `First make it correct, then make it fast.` .Cp er jonno jeita must mone rakhte hobe >> 
Problem -> Brute Force ভাবো -> এটা কি Correct? -> হ্যাঁ -> এখন Time Complexity বের করো -> Slow? -> হ্যাঁ -> Optimization খুঁজো

5. Dry Run কী এবং কেন দরকার?
=> Dry Run holo code korar por sompurno code ta nije mathay ba mone mone miliye dekha sob thik kaj korche naki, echara o khatay code run kore check kore dekha jay. Ete kon code er output kemon hobe seita bujar khotoma onek duruto bare. Ekta input nilam loop gurlo dekhlam ki change hoilo , abar nilam dekhlam ki change hoilo and ami ja cacchi ami tai pacchi naki, eivabei dry run kore.
